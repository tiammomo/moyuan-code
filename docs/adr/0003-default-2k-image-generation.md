# ADR-0003 图像生成默认 2K

状态：accepted
日期：2026-05-03
决策者：architect

## 背景

Moyuan 使用 `gpt-image-2` 辅助生成架构图、流程图、部署拓扑图和讲解资产。之前测试中 `3072x2048` 可以成功生成，`4096x2160` 探测受服务端配额限制，无法确认稳定性。

## 决策

项目内生图默认使用横版 2K，尺寸为 `3072x2048`。4K 只作为显式实验参数或本地后处理目标，不作为默认生成尺寸。

## 影响

- 生图脚本默认 `IMAGE_SIZE=3072x2048`。
- docs 中的 Image Adapter 明确默认 2K。
- 架构图只作为辅助讲解，不作为事实来源。
- 图片 prompt 必须脱敏。

## 替代方案

- 默认 1536x1024：稳定但细节不足。
- 默认 4K：理论效果更好，但服务端支持、成本和稳定性不确定。
